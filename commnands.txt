Create a new Helm Chart named todo:
helm create todo

Install it (from the todo folder):
helm upgrade -i todo ./

Update the chart (after few changes in the values.yaml file):
helm upgrade -i todo ./

In case we need to rollback to previous version:
helm rollback todo

Delete it:
helm delete todo
